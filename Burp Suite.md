## shortcuts

## Settings
### Do this when you want to "reset everything":

    Proxy > HTTP History

        Right-click > Clear history

    Target > Site map

        Right-click in the left panel (host list) > Remove all

    Repeater / Intruder / Decoder / Comparer

        Right-click on each open tab > Close tab
